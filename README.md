## Lesson 01: Why React?

### What is Declarative Code?

Is the following code imperative or declarative?

```html
const people = ['Amanda', 'Geoff', 'Michael', 'Richard', 'Ryan', 'Tyler']
const longNames = people.filter(name => name.length > 6)
```

SOLUTION:
Declarative
